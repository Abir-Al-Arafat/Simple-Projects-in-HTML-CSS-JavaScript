Frequently Asked Questions

What is object-oriented Programming?

a programming paradigm that uses objects and classes in programming.

What are the main concepts of Object-Oriented Programming (OOPs)?

Class, Objects, Polymorphism, Encapsulation, Inheritance, Abstraction

What is a classs?

A class contains the blueprints or the prototype from which the objects are being created.

What is object?

The object is an entity that has a state and behavior associated with it.

What is Inheritance?

Inheritance is the capability of one class to derive or inherit the properties from another class.

What is Polymorphism?

Polymorphism simply means having many forms.

What is Encapsulation?

The idea of wrapping data and the methods that work on data within one unit.

What is Abstraction?

Abstraction hides the unnecessary code details from the user.
